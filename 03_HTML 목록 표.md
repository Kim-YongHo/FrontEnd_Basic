# 목록

---

## <**ol**>, <**li**> (순서)

- Ordered List

- 순서를 기준으로 목록 생성

- `type = "a",  type = "A"` 등을 추가해서 사용 



## <**ul**>, <**li**> (순서X)

- unordered List

- 순서 없이 목록 생성
  
  

## <**dl**>, <**dt**>, <**dd**>

- 설명 목록 생성

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>TEST page</title>
</head>
<body>

    <ol type = "a">
        <li>당근</li>
        <li>중독</li>
        <li>레고</li>
    </ol>

    <ul>
        <li>당근</li>
        <li>중독</li>
        <li>레고</li>
    </ul>

    <dl>
        <dt>당근 구매 내역</dt>
        <dd>레고</dd>
        <dd>카트리지</dd>
    </dl>

    <dl>
        <dt>대통령 후보</dt>
        <dd>이재명</dd>
        <dd>윤석열</dd>
    </dl>
    
</body>
</html>
```
